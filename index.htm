./about

Find Security Bugs is plugin for FindBugs that aim to help security audit on Java web application.

./download

Get the plugin v 0.1 Get Find Bugs

./bugs

The bug patterns identify by the plugin are not automatically a vulnerability or a defect. It represent sensible points of the application that should be analyse closely.

  • Predictable Pseudo Random Generator (PRG)
  • Servlet parameter
  • Request Content-Type
  • Request Hostname (ServerName/Host)
  • Request Session Id
  • Request Query String
  • Request Header
  • Request Header "Referer"
  • Request Header "User-Agent"
  • Request Header
  • Cookie usage
  • Path traversal (read file)
  • Path traversal (write file)
  • Command Injection
  • FilenameUtils partial filtering
  • Weak TrustManager implementation
  • JAX-WS (JSR224) endpoint
  • JAX-RS (JSR311) endpoint
  • Tapestry Page
  • Wicket WebPage
  • Weak MessageDigest
